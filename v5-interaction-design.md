# 护苗行动 v5.0 · 安全岛交互设计文档

## 一、核心创意

**一句话**：孩子进入一座"安全岛"，岛上不同区域有小朋友正要做危险的事，孩子需要用安全知识说服他们远离危险。

**从"做题"到"说服"的范式转变**：
- 不是选ABC，而是用自然语言/语音和AI角色对话
- AI角色有真实性格（倔强、好奇、爱面子），会根据孩子的说服方式动态回应
- 成功说服后获得勋章，失败会激发重试欲望

## 二、页面结构（4页）

### 页面1：安全岛地图（核心创新页）
- CSS视差滚动背景（海洋+岛屿）
- 5个可点击区域热点（脉冲动画）
- 已完成的区域显示勋章，未完成的显示"?"
- 点击区域→缩放过渡动画→进入场景

### 页面2：场景对话（核心体验页）
- 场景背景（河流/海滩/泳池/水库/工地）
- 小朋友角色卡片（头像+名字+性格标签）
- 小朋友正在做危险的事（动画表现）
- 对话窗口：孩子说→AI角色回应→循环
- 说服进度条（可视化）
- 成功/失败动画

### 页面3：首页（角色创建）
- 数据震撼开场（溺水统计）
- 输入名字
- 选择年龄（影响场景复杂度）

### 页面4：成长档案
- 安全积分
- 已拯救的小朋友
- 勋章墙
- AI教练评语
- 证书生成

## 三、角色设计

| 场景 | 角色 | 性格 | 危险行为 | 说服难度 |
|------|------|------|---------|---------|
| 河边 | 小刚 | 调皮、爱逞强 | "河水好凉快，下来游泳！" | 中 |
| 海滩 | 小红 | 好奇、容易兴奋 | "浪来了！冲进去好玩！" | 低 |
| 泳池 | 小明 | 爱面子、怕被嘲笑 | "敢不敢去深水区？胆小鬼！" | 高 |
| 水库 | 小芳 | 天真、容易被怂恿 | "这里没人管，可以随便玩！" | 中 |
| 工地 | 小强 | 贪玩、不顾危险 | "泥水坑好好玩，快来踩！" | 低 |

## 四、说服机制

### 对话流程
```
1. 孩子进入场景，看到角色正在做危险的事
2. 角色主动说话（语音播报）："快来啊，水好凉快！"
3. 孩子输入/语音："不要下去，野外河流有暗流！"
4. AI评估孩子的回答：
   - 包含正确安全知识→角色被说服→成功
   - 理由不充分→角色继续倔强→需要更好的理由
   - 完全错误→角色更想尝试了→危险升级
5. 最多3轮对话，3轮后自动判定
```

### AI评估System Prompt
```
你正在扮演一个[角色性格]的小朋友[名字]。
[名字]正要做一件危险的事：[危险行为描述]。

另一个小朋友正在试图说服[名字]不要去做。

请根据对方的说服内容判断：
1. 如果对方提到了正确的安全知识（[具体知识点]），[名字]被说服了，表现出害怕/理解，并感谢对方
2. 如果对方的理由不充分，[名字]继续[性格表现]，要求更好的理由
3. 如果对方说错了，[名字]更想尝试了，危险升级

每次回复保持角色性格，用口语化、儿童能理解的语言。
回复格式：{"persuaded": true/false, "reply": "角色回应", "progress": 0-100}
```

## 五、视觉设计

### 安全岛地图
- 背景：CSS视差海洋（保留v4.0的Canvas海洋）
- 岛屿：SVG绘制的卡通岛屿
- 热点：脉冲光圈动画，hover放大
- 区域标记：emoji图标+区域名称

### 场景页面
- 背景：场景主题色渐变
- 角色：圆形头像+表情动画
- 对话气泡：角色在左（白色），孩子在右（蓝色）
- 说服进度条：底部渐变色条
